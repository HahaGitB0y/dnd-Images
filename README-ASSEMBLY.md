# Dead World Professions - Assembly Guide

## File Order for Homebrewery

Copy and paste these files **in order** into a single Homebrewery document:

1. `01-intro-learning-system.md` — Cover page, learning system rules, level progression
2. `02-medical-professions.md` — First Responder, Doctor, Nurse, Pharmacist, Dentist, Veterinarian, Psychologist
3. `03-engineering-professions.md` — Mechanic, Electrician, Engineer, Carpenter, Welder, Plumber, IT Specialist, Radio Operator
4. `04-culinary-agricultural.md` — Cook, Farmer, Botanist, Hunter, Tanner, Brewer, Fisher
5. `05-science-technical.md` — Chemist, Biologist, Explosives Expert, Surveyor
6. `06-combat-professions.md` — Marksman, Melee Combatant, Martial Artist, Military Tactician, Survivalist, Gunsmith, Archer, Knife Fighter, Shield Bearer, Breacher
7. `07-social-leadership.md` — Teacher, Leader, Negotiator, Merchant, Entertainer, Locksmith
8. `08-trade-utility.md` — Tailor, Driver, Scavenger, Combat Medic, Firefighter, Sailor, Animal Handler, Lumberjack, Miner, Small Engine Mechanic
9. `09-specialized-professions.md` — Paramedic, Optician, Athlete, Pilot, Spy, Pyrotechnics, Heavy Equipment, Mortician
10. `10-appendices.md` — Quick-reference professions, synergies, starting packages, material locations, DM guidelines

## Total Professions: 60+

## Adding Pictures

After pasting into Homebrewery, add images using:
```
{{imageMaskEdge5,--pointed,--bgc-#1a1a1a
![](YOUR_IMAGE_URL_HERE){height:100%}
}}
```

Or for full-page art:
```
{{fullPageArt
![](YOUR_IMAGE_URL_HERE)
}}
```

## Notes

- Every `\page` creates a new page in the PDF output
- The `{{classTable,wide}}` blocks create styled class tables
- You can add art between any `\page` breaks
- The `{{quote}}` blocks create styled quote boxes for flavor text
